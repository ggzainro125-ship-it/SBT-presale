# SBT Presale

A full-stack application for SBT (Soulbound Token) presale with React frontend and Rust backend.

## Project Structure

- `frontend/` - React application for the user interface
- `backend/` - Rust backend server
- `SETUP_GUIDE.md` - Detailed setup instructions

## Getting Started

Please refer to the `SETUP_GUIDE.md` for detailed setup and installation instructions.

## Technologies Used

- **Frontend**: React, JavaScript/TypeScript
- **Backend**: Rust
- **Version Control**: Git
